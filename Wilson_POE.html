Principles of Engineering

Project: The Pen Pal Machine (P1.1.6 Compound Machine)

Course: Principles of Engineering (POE)

Duration: 8/29/2018 - 9/7/2018

Objective: Make a complicated text based game using python.

Type of Project: Pair Programming Project

This was the first python project experienced in CSE. We started off with basic instructions and an objective. Our objective was to make a complicated game or interactive animation on Scratch. We spent two weeks adding sprites, refining movements, and making sure there was no bugs. Bugs being our biggest problem that took trial and error to fix. In that time, I learned discussion and bringing in other ideas is key to success. My contribution came in writing half the code and half of the documentation.
