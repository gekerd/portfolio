Project: Scratch Project

Course: CSE (Computer Science Engineering)

Duration: About 2 weeks (Late August - Early September)

Objective: Make a complicated game or interactive animation on Scratch using the taught variables

Type of Project: Pair Programming Project

This was the first project experienced in CSE. We started off with basic instructions and an objective. Our objective was to make a complicated game or interactive animation on Scratch.
