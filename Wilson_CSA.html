AP Computer Science Applications

Project: Interactive Seating Chart

Course: AP Computer Science Applications (CSA)

Duration: 8/15/2019 - 9/06/2019

Objective: Add yourself to a class interactive seating chart using Greenfoot

Type of Project: Individual Project

This was the first project experienced in CSA. I started off by looking through the rubric. Then, I decided that I wanted my image to bounce across the screen for my walking animation. In making this animation, I learned how to use the Math.random() method in Java as I made my image have random y-coordinates while moving. For the x coordinates, I learned how to use 2D arrays, so my image’s x-coordinates would be represented by the rows (which increased and decreased by specific amounts) and the y-coordinates would be represented by the columns (based on the Math.random method). Also, I used if-then and else statements, so my image could answer questions and also so some questions could lead to others. Additionally, I learned java documentation formatting and became more familiar with Greenfoot.
